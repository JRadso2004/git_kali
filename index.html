Concepto de GIT

Es una forma de carpeta virtual donde podemos crear y guardar documentos sin la necesidad de ir guardando las actualizacions como copias del mismo archivo

PRINCIPALES COMANDOS DE GIT

  • Git init:Este comando funciona para inicializar un proyecto con git
  • Git add .:Permite agregar cambios al area de reparación
  • Git commit -m "COMENTARIO":Permite dejar un comentario dentro de git
  • Git status: Permite visualizar el estado actual del arichivo
  • Git log:Permite visualizar el historial de lo realizado
  • Git branch:Premite visualizar en que rama nos encontramos
  • Git -b branch "NAME":Permite cambiar el nombre de la rama en la que nos encontramos
  • Git remote add origin "url"Permitio entre lazar el GIT con el GitHub con el link que nos otorgó
  • Git config --global user.name: Nos permite otorgar un nombre al que esta editando
  • Git config --global user.email:Nos permite otorgar un correo al que esta editando
  • Git config user.name: Vemos el nombre registrado
  • Git config user.email:Vemos el correo registrado
  • Git push:Nos sube lo elaborado.
